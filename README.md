# AfriTalent

## Présentation

**AfriTalent** est un site vitrine développé dans le cadre du projet de Semestre 2. Il s'agit d'une plateforme fictive mettant en relation des freelances tech africains avec des entreprises à la recherche de compétences dans différents domaines du numérique.

L'objectif du projet est de concevoir un site web moderne, responsive et interactif en utilisant les technologies du développement web front-end.

## Fonctionnalités

Le site intègre les fonctionnalités suivantes :

* Navigation responsive avec Bootstrap
* Navbar dynamique au défilement
* Mode sombre (Dark Mode) avec sauvegarde dans le Local Storage
* Bouton « Retour en haut »
* Compteurs animés au scroll
* Animations d'apparition des sections (Fade-In)
* Filtrage dynamique des freelances par catégorie
* Validation complète du formulaire de contact
* Responsive Design (mobile, tablette et ordinateur)
